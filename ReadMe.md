# Introduction to configuration of the Microsoft Word

Goal of this project is to harness with application of Visual Basic for Applications (VBA) at least some peculiarities of Microsoft Word.

This repository gathers:
- manual: MicrosoftWord_ConfigurationManual.docx, which should explain, how to use content of this repository
- VBA code 
- other files which create Microsoft Word environment
