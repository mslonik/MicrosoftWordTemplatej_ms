#### Files:

1. "Built-In Building Blocks.dotx" is specially prepared empty file to be placed in 
c:\Users\<UserName>\AppData\Roaming\Microsoft\Document Building Blocks\1045\16\Building Blocks.dotx. 
Replace original file if needed when Microsoft Word application is switched off.

2. MicrosoftDefaultBB.dotx are all default Building Blocks of Microsoft Word coming from the default (autogenerated) "Built-In Building Blocks.dotx". Suggestion: copy this file to the following folder:
C:\Users\<UserName>\AppData\Roaming\Microsoft\Word\startup\MicrosoftDefaultBB.dotx 

3. BB_ms.dotm are customized (user defined) Building Blocks. The default (recommended) location for the file BB_ms.dotm:
C:\Users\<UserName>\AppData\Roaming\Microsoft\Word\Startup\*ms_BB.dotm

4. BB_ms_BB_Content.docx.
This file contains all definitions of the BuildingBlocks. Made by macro BB_ExportAll from module BuildingBlocks in Macros_ms.dotm.
